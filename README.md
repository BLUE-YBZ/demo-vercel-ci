# Vercel Preview Deployment Starter

[Vercel Preview Deployments](https://vercel.com/docs/deployments/preview-deployments) allows to test code changes in a live environment. This repo configures GitHub workflow that triggers preview deployments based on commits in a pull request. This is an efficient way to test changes real-time in a staging environment, and share it with collaborators.

## Installation

Clone this repository

```bash
git clone https://github.com/D-Antonelli/demo-vercel-ci.git
cd demo-vercel-ci
```

Install the required dependencies
```
npm install
```
